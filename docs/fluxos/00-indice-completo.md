# Índice Completo dos Fluxos

> **Nota:** Este é um índice de referência. Para a documentação completa e detalhada de todos os fluxos, consulte [DOCUMENTACAO_FLUXOS_COMPLETA.md](../DOCUMENTACAO_FLUXOS_COMPLETA.md)

## Fluxos do Editor Web

1. **Criação de Campo/Mapa (POST)** - Cria novo campo com mapa completo
2. **Edição de Campo/Mapa (PUT)** - Atualiza campo existente com sincronização incremental
3. **Visualização de Campo/Mapa (GET)** - Carrega e renderiza mapa em modo somente leitura
5. **Seleção de Campo (FieldSelector)** - Gerencia seleção e navegação entre campos
6. **Manipulação do Canvas** - Todas as interações do usuário (adicionar, mover, agrupar, etc.)
7. **Cálculo de Status e Cores** - Calcula cor visual do tracker baseado nas estacas

## Fluxos de Inspeção

4. **Inspeção (update_inspection)** - Atualiza resultado de inspeção (aprovada/reprovada)

## Fluxos de Integração

8. **Integração App Móvel - Inspeção** - Fluxo completo de inspeção no app móvel
9. **Integração Painel Web - Visualização** - Visualização e gerenciamento de mapas no painel
